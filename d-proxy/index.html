Docker Hub 镜像搜索


️️️ 提示：

️️ 输入要搜索镜像的关键词，按回车键快速搜索

️️️ 基本使用示例：

️️️ 在官方镜像路径前面加上本站域名

docker pull docker.arelay.cn/stilleshan/frpc:latest
docker pull docker.arelay.cn/library/nginx:stable-alpine3.19-perl
️️️ 一键设置镜像加速：

️️️ 修改文件 /etc/docker/daemon.json（如果不存在则创建）

sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json <<-'EOF'
{
  "registry-mirrors": ["docker.arelay.cn"]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker
